RuruGrab 是一款 local-first 的 Windows 应用，用于在一个地方完成收集、整理和再次查找。

它将文件管理、批量重命名、重复文件检测、网页文本提取、加密 Vault 笔记，以及跨多个驱动器的目录式浏览整合到一个工作区中。

## 功能

- **文件管理 (File Management)**
  - 基于规则整理文件
  - 批量重命名文件
  - 查找重复文件

- **OneSpace**
  - 在一个目录式视图中浏览多个驱动器中的文件

- **网页文本提取 (Web Text Extraction)**
  - 将网页文本提取为 Markdown
  - 支持后续保存与搜索已收集内容

- **Vault**
  - 仅本地的加密笔记
  - 基于主密码访问

- **浏览器配套扩展 (Browser Companion)**
  - 支持 Chrome 和 Edge
  - 将受支持的内容发送到桌面应用

## 安装

1. 打开 [Releases](../../releases/latest) 页面
2. 下载最新安装程序
3. 安装并启动 RuruGrab

### 浏览器扩展 (Browser Extension)

1. 打开 `chrome://extensions/` 或 `edge://extensions/`
2. 启用 **Developer mode**
3. 点击 **Load unpacked**
4. 选择已安装的 RuruGrab 目录中的 `extension` 文件夹

示例路径：

```text
C:\Users\<YourUserName>\AppData\Local\RuruGrab
```
