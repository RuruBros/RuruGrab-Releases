RuruGrab 是一款面向 Windows 的本地优先（local-first）应用，用来把收集、整理和再次查找整合到同一个工作流里。

它将文件管理、批量重命名、重复文件检测、网页文本提取、加密 Vault 笔记，以及跨多个磁盘的 OneSpace 浏览整合到一个工作区中。

## 主要功能

- **文件管理**
  - 按规则整理文件
  - 批量重命名文件
  - 查找重复文件

- **OneSpace**
  - 在一个界面中浏览多个磁盘上的文件
  - 更轻松地管理分散的文件

- **网页捕获（Web Capture）**
  - 将网页文本提取为 Markdown
  - 以后可以再次搜索已保存的内容

- **Vault**
  - 将私密笔记加密保存在本地
  - 通过主密码（master password）保护

- **浏览器联动**
  - 支持 Chrome 和 Edge
  - 将支持的内容发送到桌面应用

## 安装方法

### 桌面应用
1. 打开 [Releases](../../releases/latest) 页面
2. 下载最新安装程序
3. 安装并启动 RuruGrab

### 浏览器扩展
1. 打开 `chrome://extensions/` 或 `edge://extensions/`
2. 开启 **Developer mode**
3. 点击 **Load unpacked**
4. 选择已安装 RuruGrab 目录中的 `extension` 文件夹

示例路径：

```text
C:\Users\<YourUserName>\AppData\Local\RuruGrab
```
