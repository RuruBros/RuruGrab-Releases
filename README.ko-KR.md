RuruGrab은 수집, 정리, 검색을 한곳에서 처리할 수 있도록 만든 로컬 우선(local-first) Windows 앱입니다.

파일 정리(file management), 일괄 이름 변경(batch rename), 중복 파일 탐지(duplicate detection), 웹 텍스트 추출(web text extraction), 암호화된 볼트 메모(vault notes), 여러 드라이브를 아우르는 OneSpace 탐색 기능을 하나의 작업 공간(workspace)으로 묶었습니다.

## 주요 기능

- **파일 관리(File Management)**
  - 규칙 기반으로 파일 정리
  - 파일 일괄 이름 변경
  - 중복 파일 찾기

- **OneSpace**
  - 여러 드라이브의 파일을 한 화면에서 탐색
  - 흩어진 파일을 더 편하게 관리

- **웹 캡처(Web Capture)**
  - 웹 페이지의 텍스트를 Markdown으로 추출
  - 저장한 내용을 나중에 다시 검색

- **볼트(Vault)**
  - 민감한 메모를 로컬에 암호화해 저장
  - 마스터 비밀번호(master password)로 보호

- **브라우저 연동(Browser Companion)**
  - Chrome, Edge 지원
  - 지원되는 콘텐츠를 데스크톱 앱으로 전송

## 설치 방법

### 데스크톱 앱(Desktop App)
1. [Releases](../../releases/latest) 페이지를 엽니다
2. 최신 설치 파일을 다운로드합니다
3. 설치 후 RuruGrab을 실행합니다

### 브라우저 익스텐션(Browser Extension)
1. `chrome://extensions/` 또는 `edge://extensions/` 를 엽니다
2. **개발자 모드(Developer mode)** 를 켭니다
3. **Load unpacked** 를 클릭합니다
4. 설치된 RuruGrab 폴더 안의 `extension` 폴더를 선택합니다

예시 경로:

```text
C:\Users\<YourUserName>\AppData\Local\RuruGrab
```
