RuruGrab은 수집, 정리, 재검색을 한곳에서 처리할 수 있도록 만든 local-first Windows 앱입니다.

파일 관리, 일괄 이름 변경, 중복 파일 탐지, 웹 텍스트 추출, 암호화된 Vault 메모, 여러 드라이브를 아우르는 카탈로그형 탐색 기능을 하나의 워크스페이스로 제공합니다.

## 기능

- **파일 관리(File Management)**
  - 규칙 기반 파일 정리
  - 파일 일괄 이름 변경
  - 중복 파일 탐지

- **OneSpace**
  - 여러 드라이브의 파일을 하나의 카탈로그형 화면에서 탐색

- **웹 텍스트 추출(Web Text Extraction)**
  - 웹 텍스트를 Markdown으로 추출
  - 수집한 내용을 나중에 저장하고 검색 가능

- **Vault**
  - 로컬 전용 암호화 메모
  - 마스터 비밀번호 기반 접근

- **브라우저 연동(Browser Companion)**
  - Chrome, Edge 지원
  - 지원되는 콘텐츠를 데스크톱 앱으로 전송

## 설치

1. [Releases](../../releases/latest) 페이지로 이동
2. 최신 설치 파일 다운로드
3. 설치 후 RuruGrab 실행

### 브라우저 익스텐션(Browser Extension)

1. `chrome://extensions/` 또는 `edge://extensions/` 열기
2. **개발자 모드(Developer mode)** 활성화
3. **Load unpacked** 클릭
4. 설치된 RuruGrab 디렉터리 안의 `extension` 폴더 선택

예시 경로:

```text
C:\Users\<사용자이름>\AppData\Local\RuruGrab
```
